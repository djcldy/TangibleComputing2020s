Week One: Introduction

This week we introduce the course and the big ideas behind it. We will review the basics of HTML, CSS, and JavaScript and create a personal website. We will discuss GitHub and create a repository for the class


Homework Instructions

    Readings

  • "Basic Animation Aesthetics (2012) David Oreilly

    • Personal Website (2%)

  • Sign-up for a GitHub Account
  • Share your GitHub user-name with your professor immediately so they can add you to the class repository.
  • Once you have access to the class github repository, you will find a folder with your name in the 'students' folder.
  • Start your personal website. Follow these instuctions:
    • Use the template website as a starting point
    • Create a table of contents in 'index.html'. It will contain a link to a page for each of the 15 weeks of the coure.
    • Make sure to create a seperate webpage for each week. For example, week 1 would be saved in 'week_01.html'.
    • Use at least 25 different HTML tags
    • Use use at least 50 different CSS tags
    • Push to GitHub when completed. Feel free to update as many times as you like.

    • Personal or Project Statement (2%)

  • Using 300 words and two images addressing create a short post that addresses the following questions: What inspires you as a Multimedia Designer? Do you have a burning project idea you want to work on this semester? Is there something you have learned at DIDI or another school you want to learn better? What is your biggest fear in this course? Include this statement on your personal website. Feel free to include any relevant links to explain your ideas. We will look at this again at the end of the course.
