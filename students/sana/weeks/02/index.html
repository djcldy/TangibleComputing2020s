Super Mario Game

Super Mario game was one of the most scuccesful video games in the '80s. The game design is

really simple since the main idea behind the game is to get from the start of the stage to the end.

The game consists of very basic elements such as ;the player, creatures like turtles, happy faces, blue sky,

capsule-like background shapes, green terrains, under ground surface, square obstacles with different

heights, coins and display for collecting coins. The gameplay mechanics are very simple and this includes;

the player can leap, move backwards, forwards and slide down, the creatures can move only left and

right and the coins rotate only. Despite the simplicity of the game design and animations; super Mario

game success proved that "difficulty doesn't equal greatness " because people still playing and enjoying

the game even with all the advanced video games in the market.

2- colourful screen

3- 2D game

5- arcade game style

7- graphics are made of rectangles and squares overly to give more 3d look

8- simple animations

9- affordance: such as the height of the box indicates the length of the character jump

and flagpole always indicates the end of the game

10-fictional mushroom kingdom environment

game interaction dynamics

7- stimulating fun game-
